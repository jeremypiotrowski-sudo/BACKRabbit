## Quick Reference: Common Commands

```powershell
# Firehose device detection
backrabbit firehose detect

# Chip info
backrabbit firehose info --device COM3

# Dump GPT
backrabbit firehose printgpt --device COM3 --loader firehose.elf

# Read partition
backrabbit firehose dump --device COM3 --loader firehose.elf --partition boot_a --output boot_a.img

# Write partition
backrabbit firehose flash --device COM3 --loader firehose.elf --partition boot_a --input stock_boot.img

# Full rescue (with TUI firmware sourcing if no --backup)
backrabbit firehose rescue full --device COM3 --loader firehose.elf

# Full rescue (with pre-specified firmware)
backrabbit firehose rescue full --device COM3 --loader firehose.elf --backup ./stock-firmware

# Firmware sourcing only
backrabbit firmware source --model SM-F966U1 --region XAA --skip-tui

# Firmware extraction from .tar.md5
backrabbit firmware extract input.tar.md5 --output ./extracted
```
